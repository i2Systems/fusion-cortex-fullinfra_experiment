# Design Tokens

This page documents all design tokens used in the Fusion/Cortex application. All design values are defined as CSS custom properties (variables) for easy theming and consistency.

## Colors

### Backgrounds
- `--color-bg`: Main background color
- `--color-bg-elevated`: Elevated background (cards, modals)
- `--color-surface`: Surface color (glass morphism)
- `--color-surface-subtle`: Subtle surface color

### Primary Colors
- `--color-primary`: Primary brand color (cyan/blue neon)
- `--color-primary-soft`: Soft primary color
- `--color-primary-hover`: Primary hover state
- `--color-primary-strong`: Strong primary color

### Accent & Status Colors
- `--color-accent`: Accent color (magenta)
- `--color-danger`: Danger/error color (hot pink)
- `--color-success`: Success color (neon green)
- `--color-warning`: Warning color (bright yellow)

### Text Colors
- `--color-text`: Primary text color
- `--color-text-muted`: Muted text color
- `--color-text-soft`: Soft text color
- `--color-text-on-primary`: Text color on primary background

### Borders
- `--color-border-subtle`: Subtle border color
- `--color-border-strong`: Strong border color

## Spacing

All spacing values scale with root font-size:

- `--space-0`: 0
- `--space-1`: 0.25rem (4px)
- `--space-2`: 0.5rem (8px)
- `--space-3`: 0.75rem (12px)
- `--space-4`: 1rem (16px)
- `--space-5`: 1.25rem (20px)
- `--space-6`: 1.5rem (24px)
- `--space-8`: 2rem (32px)
- `--space-10`: 2.5rem (40px)
- `--space-12`: 3rem (48px)
- `--space-16`: 4rem (64px)
- `--space-20`: 5rem (80px)

## Border Radius

- `--radius-xs`: 4px
- `--radius-sm`: 6px
- `--radius-md`: 8px
- `--radius-lg`: 12px
- `--radius-xl`: 18px
- `--radius-2xl`: 24px
- `--radius-pill`: 999px

## Shadows & Glows

- `--shadow-soft`: Soft shadow
- `--shadow-md`: Medium shadow
- `--shadow-strong`: Strong shadow
- `--shadow-glow-primary`: Primary glow effect
- `--shadow-glow-accent`: Accent glow effect
- `--shadow-glow-hover`: Hover glow effect

## Usage Example

```tsx
<div 
  className="p-[var(--space-4)] rounded-[var(--radius-lg)]"
  style={{
    backgroundColor: 'var(--color-surface)',
    border: '1px solid var(--color-border-subtle)',
    boxShadow: 'var(--shadow-soft)',
  }}
>
  Content using design tokens
</div>
```
